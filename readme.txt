Easily manage quantity step increments for WooCommerce products and variants with built-in validation.

== Description ==

WC Product Quantity Steps enhances your WooCommerce store by allowing you to set custom quantity increments for products and their variants. This plugin helps you maintain better inventory control and improve the shopping experience.

= Key Features =

* Set custom quantity steps for individual products
* Configure different steps for product variants
* Client-side validation to prevent invalid quantities
* Easy-to-use admin interface in product settings
* Compatible with WooCommerce bulk edit
* Lightweight and optimized for performance

= How It Works =

The plugin adds a new field to your product settings where you can specify the quantity step increment. For example, if you set a step of 5, customers can only add products in quantities of 5, 10, 15, etc.

= Use Cases =

* Wholesale products that must be ordered in specific quantities
* Bulk items that are always sold in fixed amounts
* Products with packaging constraints
* Service packages with predefined quantities

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to any product in WooCommerce
4. Find the 'Quantity Steps' field in the product data panel
5. Set your desired quantity step value

== Frequently Asked Questions ==

= Does this work with variable products? =

Yes, you can set different quantity steps for each variant of a variable product.

= Can I set a different step for each product? =

Yes, quantity steps can be configured individually for each product or variant.

= Is it compatible with the latest WooCommerce version? =

Yes, the plugin is regularly tested and updated to maintain compatibility with the latest WooCommerce releases.
